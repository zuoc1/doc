# 1. 文档内跳转

按住Ctrl左键点击链接

### <a id="table1">方式一</a>

```tex
<a id="table1">Table - 1</a>    #要跳转的位置,id = 链接位置
[跳转到 Table-1](#table1)        #在需要跳转的地方引用,点击即可实现跳转(编辑器中右键打开链接)
```

### <a id="table2">方式二</a>

```tex
<a id="table2">Table - 2</a> 		#要跳转的位置,id = 链接位置
<a href="#table2">跳转到 Table-2</a>	#在需要跳转的地方引用,点击即可实现跳转(编辑器中右键打开链接)
```
